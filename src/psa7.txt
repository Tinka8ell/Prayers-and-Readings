   <div class="poetry top-1">
    <p class="line">
     <span class="text Ps-7-1">

      </span>

     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-1">
       Save me from my persecutors—rescue me!
      </span>
     </span>
     <br/>
     <span class="text Ps-7-2" id="en-NLT-13974">
      <sup class="versenum">
       2
      </sup>

     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-2">
       tearing me to pieces with no one to rescue me.
      </span>
     </span>
     <br/>
     <span class="text Ps-7-3" id="en-NLT-13975">
      <sup class="versenum">
       3
      </sup>
      O
      <span class="small-caps" style="font-variant: small-caps">
       Lord
      </span>
      my God, if I have done wrong
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-3">
       or am guilty of injustice,
      </span>
     </span>
     <br/>
     <span class="text Ps-7-4" id="en-NLT-13976">
      <sup class="versenum">
       4
      </sup>
      if I have betrayed a friend
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-4">
       or plundered my enemy without cause,
      </span>
     </span>
     <br/>
     <span class="text Ps-7-5" id="en-NLT-13977">
      <sup class="versenum">
       5
      </sup>
      then let my enemies capture me.
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-5">
       Let them trample me into the ground
      </span>
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-5">
       and drag my honor in the dust.
       <span class="selah">
        Interlude
       </span>
      </span>
     </span>
    </p>

      Arise, O
      <span class="small-caps" style="font-variant: small-caps">
       Lord
      </span>
      , in anger!
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-6">
       Stand up against the fury of my enemies!

     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-6">
       Wake up, my God, and bring justice!
      </span>
     </span>
     <br/>
     <span class="text Ps-7-7" id="en-NLT-13979">
      <sup class="versenum">
       7
      </sup>
      Gather the nations before you.
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-7">
       Rule over them from on high.
      </span>
     </span>
     <br/>
     <span class="indent-1">
      <span class="text Ps-7-8" id="en-NLT-13980">
       <sup class="versenum">
        8
       </sup>
       <span class="indent-1-breaks">
       </span>
       The
       <span class="small-caps" style="font-variant: small-caps">
        Lord
       </span>
       judges the nations.
      </span>
     </span>
     <br/>
     <span class="text Ps-7-8">
      Declare me righteous, O
      <span class="small-caps" style="font-variant: small-caps">
       Lord
      </span>
      ,
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-8">
       for I am innocent, O Most High!
      </span>
     </span>
     <br/>
     <span class="text Ps-7-9" id="en-NLT-13981">
      <sup class="versenum">
       9
      </sup>
      End the evil of those who are wicked,
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-9">
       and defend the righteous.
      </span>
     </span>
     <br/>
     <span class="text Ps-7-9">
      For you look deep within the mind and heart,
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-9">
       O righteous God.
      </span>
     </span>
    </p>
   </div>

      <span class="indent-1-breaks">
      </span>

     </span>
     <br/>
     <span class="text Ps-7-11" id="en-NLT-13983">
      <sup class="versenum">
       11
      </sup>
      God is an honest judge.
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-11">
       He is angry with the wicked every day.
      </span>
     </span>
    </p>

       12
      </sup>
      If a person does not repent,

       God

        ]

     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>

       he will bend and string his bow.
      </span>
     </span>
     <br/>
     <span class="text Ps-7-13" id="en-NLT-13985">
      <sup class="versenum">
       13
      </sup>

     </span>

     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>

       and shoot his flaming arrows.

     </span>
    </p>

     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-14">
       they are pregnant with trouble
      </span>
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-14">

      </span>
     </span>
     <br/>
     <span class="text Ps-7-15" id="en-NLT-13987">
      <sup class="versenum">
       15
      </sup>
      They dig a deep pit to trap others,
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-15">
       then fall into it themselves.
      </span>
     </span>
     <br/>
     <span class="text Ps-7-16" id="en-NLT-13988">
      <sup class="versenum">
       16
      </sup>
      The trouble they make for others backfires on them.
     </span>
     <br/>
     <span class="indent-1">
      <span class="indent-1-breaks">
      </span>
      <span class="text Ps-7-16">
       The violence they plan falls on their own heads.
      </span>
     </span>
    </p>
